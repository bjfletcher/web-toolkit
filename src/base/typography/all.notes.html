  • You can optionally add a smaller class to the headers too
  • By default text is font-family: skyTextRegular, helvetica, arial.
  • To make something font-family: SkyTextMedium, helvetica, arial we would prefer you use the class medium.
  • To use font-family: luna, helvetica, arial we would prefer you use the class instructions.
